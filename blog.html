Question-1: What are the different ways to select an element in the DOM?

There are many ways to select an element it the DOM! Some of the common ways are given below: 1. getElementById("myElementId") 2. getElementsByClassName("myClassName") 3. getElementsByTagName("div") 4. querySelector(".myClassName") 5. querySelectorAll(".myClassName") etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

1. innerHTML: We can get and set the html content of an element including html tags. It can render html tags. 2. innerText: It can not render html tags. We can use it to get or set plain text content of an element. It ignores hidden elements. 3. textContent: Can not render html tags. It can get or set text content of an element including hidden text.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in JavaScript. When a single event listener is added to a parent to handle event for multiples child elements even if they are added dynamically, is called Event delegation.

Question-4: What is event bubbling in the DOM?

Event bubbling is a technique where an event starts from a targeted element and bubbles up trough its parent elements until it reaches the root.

Question-5: How do you create, add, and remove elements using JavaScript?

1. Creating an Element: We can use document.createElement(tagName) to create elements. 2. Adding Elements to the DOM: We can use .appendChild() to add the newly created elements. 3. Removing Elements from the DOM: We can use .remove() to remove an element from the dom.
